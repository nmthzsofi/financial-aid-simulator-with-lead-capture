# Financial Aid Eligibility Simulator

An interactive Forward College eligibility tool that gives prospective students an indicative financial-aid range based on household income, country, academic performance, programme choice, and study costs.

## Highlights

- Implements a multi-step, data-driven eligibility calculation using four CSV datasets
- Dynamically adapts diploma, grade, country, and programme options to user selections
- Validates applicant inputs and presents minimum and maximum aid estimates
- Captures qualified leads through the HubSpot Forms API

## Tech Stack

JavaScript, HTML5, CSS3, CSV, Fetch API, and HubSpot Forms API.
